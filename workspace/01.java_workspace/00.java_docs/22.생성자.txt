 
## 생성자 ##

1. 특징
	-- new 객체 생성시에 자동 호출 수행
	-- 주로 멤버변수 초기화 수행에 사용
	-- 또는 객체가 서비스(사용)하기 전에 선행처리 되어야할 로직 수행
	-- (배열등 자료저장소 생성, DBMS드라이버 로딩등)
	-- 공통 처리 로직
	-- 중복 정의 가능 : 생성자의 아규먼트(타입, 순서, 갯수)가 다르게 다중 선언
	-- 모든 클래스는 최소 1개 이상의 생성자가 존재해야함(규칙)
	
2. 생성자 선언 형식 :
	-- Access_Modifier 클래스이름() { }
	-- Access_Modifier 클래스이름(타입 매개변수명, 타입 매개변수명) { }	
	-- 생성자에는 반환타입을 표기해서는 안됨

3. 기본생성자
	-- 개발자가 명시적으로 생성자를 선언하지 않는경우에
	-- 컴파일러가 자동으로 제공하는 생성자
	-- 기본생성자 형식: public 클래스이름(){}
	-- no argument, empty body
	-- 특별하지 않는 경우에 생성자 중복정의시에 기본생성자도 중복정의를 권장 (상속설명)
