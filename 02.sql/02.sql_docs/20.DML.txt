
## ========================

	DML (Data Manuplation Languange) : 데이터 조작어

    -- 레코드 추가(insert), 변경(update), 삭제(delete), 조회(select)
    -- COMMIT / ROLLBACK 명시적 지정해야함
	

	1. 레코드 추가
		(1) 지정한 컬럼에 대해서 값을 순서데로 매핑해서 레코드 추가, not null 컬럼 반드시 포함
			INSERT INTO 테이블명(컬럼명1, 컬럼명N) VALUES(값1, 값N);

		(2) 컬럼명을 생략하면 테이블 구조 컬럼순서데로 모두 차례데로 값을 부여 추가
			INSERT INTO 테이블명 VALUES(값1, 값2, 값N);

		(3) 다중레코드 추가 : 기존테이블 레코드 데이터를 참조 추가
			INSERT INTO 테이블명(컬럼명, 컬럼명X)
			SELECT 컬럼명, 컬럼명X FROM 테이블명 WHERE 조건문
			;

	2. 레코드 변경
	(1) 지정한 컬럼에 대해서 모든 레코드의 값을 변경
		UPDATE 테이블명 SET 컬럼명1=변경값 , 컬럼명N=값;

	(2) 지정한 조건을 만족하는 레코드의 컬럼에 대해서 값을 변경
		UPDATE 테이블명 SET 컬럼명1=변경값 , 컬럼명N=값 
		WHERE 조건구문;
