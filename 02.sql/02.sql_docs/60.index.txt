	## INDEX Object 

	--	명령문의 처리속도를 향상하기 위히 컬럼에 적용하는 Object
	--	Primry key 컬럼에는 기본적으로 인덱스 객체 생성됨


	1. 생성:
		CREATE INDEX 인덱스명
		ON 테이블명 (컬럼명 정렬방법, 컬럼명N 정렬방법)
		;

	2. 삭제 
		DROP INDEX 인덱스명;

	3. DICTIONARY
		DESC USER_INDEXES;
		SELECT index_name, table_name FROM user_indexes;
