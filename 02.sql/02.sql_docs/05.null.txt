
/** null 컬럼

-- 어떠한 데이터도 갖고 있지 않음을 의미
-- 연산을 수행할 수 없음 (이유, NULL은 데이터가 아님)
-- 0은 숫자데이터, 공백문자는 문자데이터임
-- 값지정시 명시적 null,  해당칼럼을 지정하지 않거나

-- ## null 컬럼에 대한 정렬조회시 null 값위치
1) ASC(올림차순)
	: 작은값 => 큰값 => null
		
2) DESC(내림차순)
	: null => 큰값 => 작은값
	
-- ## null 컬럼에 대한 조회
1) 컬럼의데이터가 null 인 레코드 조회 
	- 컬럼명 is null 
	
2) 컬럼의 데이터가 null이 아닌 레코드 조회
	- 컬럼명 is not null 
	- 컬럼명 not in (null)

-- ## null 컬럼에 대한 연산은 수행하지 않으므로 
-- 따라서 , null을 임의 데이터로 대체하여  연산을 수행 가능

-- ## null 컬럼에 대한 지정한 값으로 대체
-- nvl(컬럼명, null대체값)
-- nvl2(컬럼명, not-null대체값, null대체값)

-- ## count(컬럼명)  수행시 null 값 행에대해서는 count하지 않음.

-- 레코드 추가시에 null 값에 대한 데이터 부여방법
-- 명시적으로 null 
-- 해당 컬럼의 데이터를 부여하지 않음

*/
