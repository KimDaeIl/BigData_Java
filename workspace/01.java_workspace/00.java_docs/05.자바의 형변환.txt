
/* 자바의 형변환 */

1. 자동 형변환 : 
	- 작은타입의 값은 큰타입의 기억공간에 자동할당
	- byte => short, char => int => long => float => double
	
	- 주의 : boolean은 형변환 발생하지 않음

2. 명시적 형변환
	- 큰타입의 값이기는 하지만 작은타입으로 형변환시 저장가능 : double, float => int 타입
	- 단, int  => byte, short 기억공간에 자동 형변환되어 초기값 할당가능함 
		(연산시는 예외, 명시적형변환필요)
	- (이유? 컴퓨터의 기본연산은 int)
	- 주의: 작은 타입의 기억공간 범위의 값이 아닌 경우에는 데이터의 손실 발생
	
	- 형식: 타입 변수명 = (타입)값;

3. 형변환은 기본형, 객체형(상속, 다형성) 모두 가능함
	 
4. 같은 유형의 연산의 결과는 같은 타입
	다른 유형의 연산의 결과는 큰타입	 
	
5. 할당연산자
	타입 변수명 = 값;
	(1) 동일타입이면 자동할당
	(2) 큰타입이면 자동할당 (메모리 낭비..아깝지만..그냥..써..요..)
	(3) 다른타입이면 컴파일 오류발생
	
	(4) 명시적 형변환:
		작은타입에 큰타입의 값을 작은타입으로 변환해서 할당가능
