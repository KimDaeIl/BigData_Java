## 클래스 설계 방법 ##

1. 클래스 선언
	-- 형식 :
	modifiers class 클래스이름 {

	}

2. 멤버변수 선언
	-- 형식 :
	-- modifiers 타입 변수명;
	-- modifiers 타입 변수명 = 명시적초기값;

3. 멤버메서드 선언
	-- 형식 :
	-- modifiers 반환타입 메서드이름(타입 매개변수명, 타입 매개변수명) {
	        수행구문;

                     // 반환타입이 있는 경우에는 반드시 해당타입의 값을 반환해야함                 
	        return 값; 

	         // 반환타입이 void 인 경우에는 return; 구문은 필요시에 선택사용	
	    }
	    
4. 생성자 선언
	-- 형식 :
	-- Access_Modifier 클래스이름(){}
	-- Access_Modifier 클래스이름(타입 매개변수명, 타입 매개변수명){}	
	-- 중복 정의 가능
