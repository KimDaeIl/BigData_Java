
## 변수의 종류 ##

1. 멤버변수
	1) 인스턴스 변수 (instance variable) 
		-- 객체 생성후 사용 가능
		-- same class 에서는 직접 변수명 사용 가능
		-- other class 에서는 객체 생성후 참조변수를 통해서 사용 가능

	2) 클래스 변수 (class variable) / 스태틱변수 (static variable)
		-- 객체 생성없이 사용 가능
		-- same class 에서는 변수명 사용가능
		-- other class 에서는 클래스이름.클래스변수명을 통해서 사용 가능

2. 지역변수
	-- 메서드 내부, 메서드의 매개변수, 생성자 내부, 제어문의 내부 선언한 변수
	-- 해당 영역 { } 내부에서 제한적 사용 가능
